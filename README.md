## ✨ Возможности

| Категория | Что умеет |
|-----------|-----------|
| **🌐 Языки** | Обнаружение установленных версий Python, Node.js, Java, Rust, C++. Установка через winget, Chocolatey, Scoop или открытие страницы загрузки |
| **📦 Пакеты** | Поиск, установка и удаление пакетов для **pip**, **npm**, **cargo**. Встроенный каталог популярных пакетов |
| **📁 Шаблоны** | Создание проектов из готовых TOML-шаблонов (CLI-заготовка, базовый проект и т.д.) |
| **🔧 Диагностика** | Информация об ОС, переменных окружения, PATH, дисках, установленных версиях |
| **🌍 Локализация** | Русский и английский интерфейс, переключение по F2 |
| **🎨 Анимации** | Плавное появление элементов, подсветка активных пунктов, цветовая тема "тёмно-синий хакер" |

## 🚀 Быстрый старт

```bash
# 1. Установите зависимости
pip install textual rich

# 2. Запустите CodeKit
python main.py

# Или просто запустите run.bat
```

**Требования:** Python 3.11+ (используется `tomllib` из стандартной библиотеки)

## ⌨️ Управление

| Клавиша | Действие |
|---------|----------|
| `↑` `↓` | Навигация по меню |
| `Enter` | Выбрать пункт |
| `Esc` | Назад / Выход |
| `F2` | Переключить язык (RU/EN) |

## 🛠️ Технологии

- **[Textual](https://textual.textualize.io/)** — фреймворк для терминальных UI
- **[Rich](https://rich.readthedocs.io/)** — форматирование текста в терминале
- **TOML** — конфигурация и локализация (встроенный `tomllib`)
- **pyfiglet** — генерация ASCII-логотипа
